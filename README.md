# StartingPoint--Jekyll
My boilerplate for building sites/blogs powered by Jekyll.

## Installation & Usage
    bundle install
    jekyll serve --watch

_Note: Requires Ruby version 2.0.0 =>_ 


## Configuration
Edit: _config.yml
