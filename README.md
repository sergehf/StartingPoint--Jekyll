# StartingPoint--Jekyll
My boilerplate for building sites/blogs powered by Jekyll.

## Installation & Usage
    bundle install
    jekyll serve --watch

_Note: Requires Ruby version 1.9.3 =>.   
    
## Configuration
Edit: _config.yml

_Note: when editing _config.yml, you need to restart jekyll to see the changes.__
